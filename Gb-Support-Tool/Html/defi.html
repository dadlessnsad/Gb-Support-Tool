Understanding DeFi

What is DeFi

DeFi stands for decentralized finance, is built of off-blockchain technology, which means there is no need for intermediaries (Banks, Brokers, and exchanges). Using blockchain and smart contracts allowing everyone to access traditional finance instruments (buying, selling, borrowing, lending) using peer-to-peer transactions.

Most of "Defi" takes place on the Ethereum blockchain which is currently the second-largest cryptocurrency by market cap. Using smart contracts and a very unique programming language, allowing developers to make Dapps ( Decentralized Application), allowing users to execute smart contracts easily.

This mean's that for a user to access "DeFi" protocols/services they would need to have Ether ($ETH) in their wallets. Because to transact with the network, you will need to pay a Gas Fee" which is measured in Gwei a smaller unit of Ether.

Type Of Dapps

  • Decentralized exchanges (DEXs)
    • Are online exchanges that let users exchange currencies to other currencies either crypto or fiat. DEXs use peer-to-peer transactions securely without the need for intermediaries. The majority of the time DEXs don't need a KYC (know your customer).
  • Stablecoins
    • These are coins that are pegged to an asset outside of cryptocurrencies, like the U.S. dollar. using reserves and a collateral system to back up the said currency, a example is Maker Dao.
  • Lending platforms
    • Uses smart contracts to eliminate the need for a bank that manages the lending as a middle man, allowing them to charge fees.
  • WBTC/WETH
    • Because Eth is not an ERC-20 standard like bitcoin, we have to "wrap" the coins. once wrapped they are an ERC-20 standard and can interact with Defi, & Dapps.
  • Staking
    • Staking uses a PoS blockchain instead of PoW blockchains. where the miners are just validators that process the transactions and earn rewards. This is considered a "Green Blockchain"
  • Yield Farming
    • Yield farming lets you lock up funds and earn rewards during the epoch. Employing defi protocols to access various interest pools as a liquidy provider.
