MetaMask Wallet

What is MetaMask

MetaMask a Browser extension/decentralized application (Dapp), that serves as an Ether wallet, as well as the ERC-20 Token standard. Just like any other extension, it is installed the same way, on compatible browsers (ie; Firefox, Google Chrome, Brave Browser, Microsoft Edge, Android, and IOS.).

By connecting to the Etherum network using Web3, users can access a whole array of decentralized financing tools, such as Decentralized exchanges, Yield Farming, and the ever-expanding Defi (Decentralized Finance).

How to install MetaMask

As mentioned above you can install the MetaMask browser extension or Dapp, by going to their official website MetaMask.io. Select which version best suits your needs, once chosen produce with the download. When the installation is complete you can then get started. If this is your first wallet create a new wallet andseed phrase. Make a secure password then, follow the steps to write down and store seed phrases .

You will need to copy and write down a 12-word seed phrase, and store it securely. This will allow you to recover the wallet.

Once you have written down and stored your seed phrase properly, MetaMask will ask you to confirm your seed phrase by choosing the corresponding words in order.

After that is complete you can now use MetaMask in its entirety
